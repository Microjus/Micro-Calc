----
### Instalação
> **Clone o Repositório**
```bash
$ git clone https://github.com/Microjus/Micro-Calc.git
```
### Execução
> **Linha de Comando**
```bash
$ chmod +x micro-calc
$ bash micro-apache ou ./micro-apache
```
